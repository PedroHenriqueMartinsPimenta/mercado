CONFIGURAÇÃO E INSTALAÇÃO
- Instalar XAMPP e Importar Base de dados vazia
- Inserir na tabela de ano, o atual ano de intalação para utilização pelo sistema
- Copiar os arquivo para C:/program files/
- Criar atalho na tela inicial
- inserir um atalho na inicialização de programas do sistema operacional juntamento com o atalho inicialização do banco de dados
